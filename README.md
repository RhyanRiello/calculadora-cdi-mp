# 💰 Calculadora de CDI

Calculadora simples para simular rendimentos baseados no CDI.

## 🚀 Funcionalidades

* Cálculo de rendimento diário
* Cálculo mensal
* Considera percentual do CDI
* Simulação com imposto

## 🛠️ Tecnologias

* HTML
* CSS
* JavaScript

## ▶️ Como usar

1. Abra o arquivo `index.html`
2. Insira o valor investido
3. Veja o rendimento
